# Npm Commands Tricks:

1. install npm packages
  ```
  npm i <Package_name>
  ```
2. uninstall npm package
  ```
  npm uninstall <package_name>
  ```
3. If you get autoprefixer warning then goto `package.json` file and add below command.
  ```
  "overrides": {
    "autoprefixer": "10.4.5"
  }
  ```
